# أسدام الفنية للزجاج — موقع الشركة (ASDAM Technical Glass Co.)

موقع تعريفي ثابت (Static Website) لشركة أسدام الفنية للزجاج، مبني بـ HTML / CSS / JavaScript بدون أي إطار عمل (Framework)، وجاهز للنشر مباشرة على **GitHub Pages**.

## المحتوى
- صفحة رئيسية (Hero) + إحصائيات الشركة
- من نحن: نبذة، الرؤية، الرسالة، القيم
- منتجاتنا: HTG، IGU، المعالجة المسبقة، الزجاج الديكوري
- المصنع والماكينات (معرض صور قابل للسحب)
- سابقة الأعمال مع فلاتر حسب المدينة
- الشهادات والتراخيص والشركاء
- نموذج طلب عرض سعر يرسل البيانات مباشرة إلى واتساب
- زر واتساب عائم في كل الصفحات
- الشعار الرسمي للشركة (نسخة ملوّنة ونسخة بيضاء للخلفيات الداكنة)

جميع الصور مأخوذة من ملف بروفايل الشركة الرسمي (صور حقيقية لمشاريع منفّذة وماكينات المصنع).

## كيفية النشر على GitHub Pages
1. أنشئ مستودعاً (Repository) جديداً على GitHub وارفع هذه الملفات كما هي (حافظ على بنية المجلدات).
2. من إعدادات المستودع (Settings → Pages)، اختر الفرع (Branch) `main` والمجلد الجذري `/ (root)`.
3. احفظ، وانتظر دقيقة — سيصبح الموقع متاحاً على رابط مثل:
   `https://USERNAME.github.io/REPO-NAME/`

## تحديث رقم واتساب / الهاتف
رقم واتساب مستخدم في 3 مواضع: زر الهيدر، الزر العائم، ونموذج طلب عرض السعر (js/main.js).
ابحث عن `966545047777` واستبدله برقمك بصيغة دولية بدون "+" أو أصفار.

## تخصيص الألوان
جميع الألوان والخطوط معرّفة كمتغيرات CSS في أول ملف `css/styles.css` تحت `:root` لتسهيل التعديل.
